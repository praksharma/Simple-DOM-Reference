# Simple DOM Reference

**Live Demo:** https://praksharma.github.io/Simple-DOM-Reference/

---

## Why?

> I have a DOM object. What can I actually do with it?

In short, a page similar to cppreference for C++ or the API documentation for libraries like NumPy or Matplotlib.

A quick guide for those who don't use javascript on a daily basis, but want to learn the basics of DOM manipulation. This is a simple reference for beginners to understand the DOM and how to manipulate it using JavaScript.

* Every page focuses on the APIs developers actually use.
* No inheritance diagrams.
* No unnecessary browser internals.
* No overwhelming lists of obscure properties.
* Just the things you'll probably reach for while building websites.

## Design Principles

Every page should answer these questions in under two minutes:

* What is this object/property?
* When should I use it?
* What are the most common examples?
* Which elements support it?

If you can answer those questions, the page is complete.
